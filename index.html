Description

GPU hardware; CUDA; massively parallel algorithms; graphics pipeline; real-time rendering; OpenGL/WebGL; and mobile GPUs.

This is a project-intensive course along with student presentations.

Coverage of mobile GPU architectures and OpenGL ES 2; GPU-acceleration on the web with WebGL; deferred shading; and broader coverage of hardware.

Prerequisites

  • Programming for Performance
  • MATH 704 : Introduction to Computer Architecture.
  • Strong C Skills
